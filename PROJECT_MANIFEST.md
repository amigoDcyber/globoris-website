## 🚀 Quick Reference

### Installation
```bash
cd globoris-website
npm install --legacy-peer-deps
npm run dev
```

### Development
- **Dev Server**: `npm run dev` (http://localhost:3000)
- **Build**: `npm run build`
- **Production**: `npm start`

### File Locations
- **Pages**: `/app/*/page.js`
- **Components**: `/components/*.js`
- **Styles**: `/app/layout.css`
- **Config**: Root directory (*.config.js)

### Customization
- **Colors**: `tailwind.config.js`
- **Content**: Page files
- **Fonts**: `app/layout.js`
- **Icons**: Component files
